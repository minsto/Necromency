# Necromency
Nocromency is mods on 1.7.10 by  atomicstrykergrumpy im porting to 1.21.x https://atomicstryker.github.io/
Branche 1.21.10 is new version for neoforge 1,21.1
## Build / Compilation

This mod is made for Minecraft NeoForge.

### Requirements

- Java 21
- Minecraft 1.21.10
- NeoForge 1.21.10
- Gradle / Gradle Wrapper included in the project

### How to build

Clone the project:

```bash
git clone <repository-url>
cd <project-folder>

Build the mod:

./gradlew build

On Windows:

gradlew.bat build

The compiled .jar file will be generated in:

build/libs/
Development setup

For IntelliJ IDEA:

./gradlew genIntellijRuns

On Windows:

gradlew.bat genIntellijRuns

Then open the project in IntelliJ IDEA and run the Minecraft client configuration.
